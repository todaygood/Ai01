## 业界AI APP

自2023年开始，字节在AI应用层面的动作就明显加快。

- 主打Chatbot类的豆包，
- 属于AI聊天机器人创建平台的Cici和Coze， 
- AI聊天机器人创建平台ChitChop，
- AI角色创建与互动平台BagelBell 

1月16日，智谱AI就抢先发布了定制化模型生成工具GLMs，实现0代码分钟级创建基础AI应用，并同时推出多模型能力自动调用工具All Tools。

百度就已经在海外上线3款AI产品，包括定位AI相机的“Meiya”，定位AI社交的“SynClub”等。
